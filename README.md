## 🚀 About

**vadims-vibe** is a pet portfolio website. Every line of code, every design decision, and every piece of content was generated and iterated with the help of AI (GitHub Copilot, ChatGPT, and friends). The goal was to see how far AI-assisted development could go — spoiler: pretty far.

The site features:

- 🎨 **Portfolio projects** — a curated showcase of frontend work
- 🖱️ **Micro-interactions** — smooth animations and delightful UX details
- 🧩 **Design system experiments** — components, tokens, and typography in action
- 📬 **Contacts page** — reach out easily

## ⚡ Getting Started

**Prerequisites:** Node.js 18+

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

Other useful commands:

```bash
npm run build   # Production build
npm run start   # Start production server
npm run lint    # Lint the codebase
```
